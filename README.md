# SentryCore

SentryCore is a highly interactive, tower-defense style plugin for Minecraft (Paper 1.21.1) that allows players to craft, deploy, customize, and upgrade powerful automatic Sentry Cores to defend their bases from hostile mobs.

## Features
- **Dynamic Firing Modes**: Fire simple Amethyst shards, continuous Prismarine tracking lasers, or devastating Echo Shard sonic booms!
- **Complete Upgrade System**: Spend **Condensed Obsidian** to upgrade Range, Damage, Recharge Speed, Emergency Healing logic, and Multi-Targeting up to 40 times.
- **Reactive Safety Nets**: Deploy the Emergency Healing stat to auto-detect incoming fatal damage and heal/buff you before it processes. Reach Tier 10 for reusable Totem of Undying resurrections!
- **Interactive GUI**: Fully customizable interfaces allowing you to swap modes, toggle targeted mobs, access internal fuel, and upgrade stats on the fly.
- **Full Data Preservation**: Packing up the Sentry Core retains all upgrades, modes, and blacklisted targets internally inside the item block data. Drop it or trade it, and it works flawlessly on placement!
